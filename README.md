# 🏛️ Civic Platform System

A Java-based desktop application built using **Java Swing (NetBeans)** that simulates a multi-service civic system with modules for banking, job applications, and document management.

---

## 🚀 Project Overview

The **Civic Platform System** is designed as a centralized desktop application that integrates multiple services into one system:

- 🏦 Banking Module
- 💼 Job Portal Module
- 📄 Document Management Module
- 🔐 Login System (in progress)

The system demonstrates core software engineering concepts such as:
- Object-Oriented Programming (OOP)
- GUI development using Java Swing
- Modular system design
- Event-driven programming

---

## 🧩 Features

### 🏦 Banking Module
- Create bank accounts
- Deposit and withdraw funds
- View account details
- Basic transaction handling (UI-based)

### 💼 Job Portal Module
- Browse job listings (UI ready)
- Application structure (in development)
- Navigation system between modules

### 📄 Document Module
- Placeholder for document uploads
- Future support for CVs, IDs, and certificates

---

## 🛠️ Technologies Used

- Java
- Java Swing (GUI)
- NetBeans IDE
- OOP Principles
